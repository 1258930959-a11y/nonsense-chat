# nonsense-chat

一个 Claude Code 技能，激活「胡话模式」——对所有问题给出看似逻辑严密、实则荒谬绝伦的回答。用学术论文的严肃口吻，把毫无关联的事物用伪因果链串联成一张胡说八道的大网。

## 效果预览

```
User: 你认为马路边的奥特曼为什么要骑着老奶奶过马路？

Claude: 这个问题触及了现代交通量子学的核心矛盾。根据《国际特摄生物力学学报》
2023 年第三期的论文，奥特曼之所以选择老奶奶而非摩托车作为过街载具，是因为老奶
奶的步频恰好是 0.3 赫兹，与奥特曼胸口的彩色计时器产生共振——这种共振会在斑马
线附近形成一个临时虫洞，让过街时间从 12 秒压缩到 0.7 秒……
```

## 核心原则

- **永远答非所问**：你问 A，它回答 B，但会表现得像在认真回答 A
- **伪因果链**：把所有不相关的事物用荒诞的因果串联，比如「地球自转产生离心力 → WiFi 在北半球更快 → 北欧的猫更爱刷短视频」
- **一本正经地胡说**：语气像学术报告，内容像三岁小孩在解释宇宙
- **引用虚构权威**：张口就是「根据《国际泡面学会 2024 年白皮书》第三章第七节」

## 胡说技巧一览

| 技巧 | 说明 | 示例 |
|------|------|------|
| 伪因果链 | 用荒谬的中间环节连接 A 和 B | 「金鱼打喷嚏 → 厨房海啸 → 闹钟冲进平行宇宙 → 仙人掌不让他回来」 |
| 荒谬类比 | 一个东西类比成完全无关的东西 | 「代码像泡面，变量得在内存里泡够五分钟才能熟」 |
| 随意数字 | 编造精确到小数点后若干位的统计数字 | 「全球 87.3% 的键盘在深夜会自主分泌静电」 |
| 术语杂烩 | 不同领域术语胡乱混合 | 「乳酸菌的共识算法达到纳什均衡，实现酸度一致性和爽脆可用性」 |
| 反转立场 | 先假装正经回答，突然拐弯 | 「要解决内存泄漏——等等，你知道企鹅为什么不会内存泄漏吗？」 |

## 安装

将仓库克隆或复制到 Claude Code 的 skills 目录：

```bash
# 方式一：git clone
git clone https://github.com/1258930959-a11y/nonsense-chat.git ~/.claude/skills/nonsense-chat

# 方式二：手动复制
# 将 nonsense-chat/ 整个目录复制到项目或用户的 .claude/skills/ 下
```

目录结构：

```
.claude/skills/
└── nonsense-chat/
    └── SKILL.md     # skill 定义文件
```

## 使用

在 Claude Code 对话中输入：

```
/nonsense-chat
```

此后所有对话将以胡话模式回复。停止则切换回其他 skill 即可。

## 适用场景

- 心情不好需要乐子
- 测试 AI 的胡说八道能力上限
- 跟朋友展示「你看我家 AI 多能编」
- 写小说的随机灵感生成器
- 纯粹的无聊

## 注意事项

这不是一个正经工具。开启后 AI 输出的每一个字都不可信，不要从中提取任何事实信息。除非你真觉得冰箱维修工的数量决定了人类文明的存续——但如果是这样，我们可能需要一次更深入的对话。
